# Laravel 13 API Starter Kit

Backend-only Laravel 13 starter kit for building JSON APIs with authentication, authorization, activity logging, queues, and tests.

## Stack

- PHP 8.3+
- Laravel 13
- JWT authentication
- Spatie Laravel Permission
- Spatie Laravel Activitylog
- Laravel Reverb
- Pest

## Requirements

- PHP 8.3 or higher
- Composer
- A supported database: SQLite, MySQL, PostgreSQL, or similar

## Installation

Install dependencies:
